Concuerror
==========

Concuerror is a systematic testing tool for concurrent Erlang programs.

How to
------

* Build Concuerror   : `make`
* Run Concuerror     : `concuerror --help`
* Run testsuite      : `make tests`
* Dialyze            : `make dialyze`
* Cleanup            : `make clean`
